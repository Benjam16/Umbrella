Deploy Umbrella marketing site (Next.js) on Vercel
=================================================

This folder is a **self-contained Next.js 15** app (App Router, Tailwind, Framer Motion).
Bento showcase + simulated terminal + Edge `/api/health-demo` for DR-style footer.

Prerequisites
-------------
- Node 20+ (matches Vercel default)

GitHub → Vercel (recommended)
------------------------------
1. Push the Umbrella repo to GitHub (github.com/Benjam16/Umbrella).
2. https://vercel.com → Add New Project → Import repository.
3. **Root Directory:** `website` (required — repo root is the CLI package).
4. Framework: Vercel auto-detects **Next.js**. Build: `npm run build`, Output: `.next` (automatic).
5. Optional env vars:
   - `NEXT_PUBLIC_UMBRELLA_API_URL` — if set, you can later change the footer fetch
     to hit your real API `GET /v1/health/dr` from a Route Handler (do not expose secrets).
6. Deploy. Custom domain: e.g. **umbrellagnt.xyz** → Project → Settings → Domains.

Local dev
---------
  cd website
  npm install
  npm run dev
  → http://localhost:3040

Production
----------
  cd website
  npm run build && npm run start

Notes
-----
- `src/data/demo-data.json` drives hero logs, DAG layout, risk demo rows, CEO briefing text.
- Edge route: `src/app/api/health-demo/route.ts` — swap for proxy to real DR endpoint when hosted API exists.
